# Portal do Olimpo

Projeto desenvolvido em React com tema de Mitologia Grega.

## Funcionalidades

- Listagem de deuses do Olimpo
- Busca por nome
- Página de detalhes de cada deus
- Navegação entre páginas

## Tecnologias utilizadas

- React
- React Router
- Styled Components
- JavaScript

## Conceitos aplicados

- Componentes
- Props
- useState
- Styled Components
- Renderização de listas

## Como executar

1. Instale as dependências:

```bash
npm install
```

2. Inicie o projeto:

```bash
npm run dev
```

3. Abra o navegador no endereço informado pelo Vite.
